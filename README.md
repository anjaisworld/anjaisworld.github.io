# The Rupeelers - Kokiri Forest Website

A responsive, animated website design inspired by The Legend of Zelda's Kokiri Forest, featuring a beautiful green color scheme and smooth animations.

## Design Features

### Color Palette (Kokiri Forest Theme)
- **Kokiri Green** (#2d5016) - Deep forest green
- **Forest Green** (#3d6b1f) - Medium green
- **Light Green** (#5a8f2f) - Bright green
- **Lime Green** (#7fb347) - Accent green
- **Gold** (#d4af37) - Rupee/accent color
- **Cream** (#f5f1e8) - Text color

## Sections

1. **Hero Section** - Welcome message with animated title and call-to-action buttons
2. **About Section** - Information about The Rupeelers with feature cards
3. **Services Section** - Four service offerings with animated icons
4. **Gallery Section** - Image grid with hover effects
5. **Contact Section** - Contact form with styled inputs
6. **Footer** - Simple footer with copyright

## How to View

Simply open `index.html` in a modern web browser. No build process or server required!

```bash
# Navigate to the directory
cd the-rupeelers

# Open in browser (macOS)
open index.html

# Or open in browser (Linux)
xdg-open index.html

# Or open in browser (Windows)
start index.html
```

## Browser Compatibility

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Customization

All colors are defined as CSS variables in `styles.css` at the top of the file. You can easily change the color scheme by modifying the `:root` variables.

## Notes

- The design uses Google Fonts (Cinzel for headings, Open Sans for body text)
- All animations are CSS-based for smooth performance
- The design is fully responsive and works on all screen sizes
- Interactive elements include hover effects and smooth scrolling
